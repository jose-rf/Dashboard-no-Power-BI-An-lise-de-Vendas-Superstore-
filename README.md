# 📊 Dashboard de Análise de Vendas – Superstore

## 🚀 Sobre o Projeto

Este projeto consiste no desenvolvimento de um dashboard interativo no Power BI com foco em análise de vendas, rentabilidade e desempenho do negócio.

O objetivo foi transformar dados brutos em insights estratégicos que apoiem a tomada de decisão, explorando diferentes perspectivas: executiva, produtos, clientes e geográfica.

---

## 🧠 Objetivos da Análise

* Monitorar o desempenho geral de vendas
* Identificar produtos e clientes mais lucrativos
* Detectar prejuízos e oportunidades de melhoria
* Analisar o impacto de descontos na rentabilidade
* Avaliar a performance por região

---

## 📊 Estrutura do Dashboard

### 📌 Visão Executiva

* KPIs de Faturamento, Lucro e Margem
* Evolução temporal dos resultados
* Indicadores de desempenho geral

---

### 📦 Análise de Produtos

* Top produtos por lucro e prejuízo
* Margem por produto
* Impacto dos descontos na lucratividade
* Identificação de produtos ineficientes

---

### 👥 Análise de Clientes

* Top clientes por faturamento e lucro
* Segmentação por tipo de cliente:

  * 🔴 Prejuízo
  * 🟡 Baixa Rentabilidade
  * 🟢 Alto Valor
* Ticket médio
* Frequência de compra
* Identificação de clientes não rentáveis

---

### 🌎 Análise Geográfica

* Distribuição de vendas por região
* Comparação de desempenho entre localidades
* Identificação de regiões com baixa performance

---

## 💡 Principais Insights

* Descontos elevados impactam negativamente a lucratividade
* Alto faturamento não garante alta rentabilidade
* Existem clientes e produtos operando com prejuízo
* Diferenças relevantes de desempenho entre regiões

---

## 🛠️ Tecnologias Utilizadas

* Power BI
* DAX (Data Analysis Expressions)
* Modelagem de Dados
* Visualização de Dados

---

## ⚙️ Principais Métricas Criadas

* Margem (%)
* Ticket Médio
* Classificação de Clientes
* Análise de Lucro e Prejuízo
